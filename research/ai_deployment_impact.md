# AI Deployment & Impact Data — All 54 African Countries

**Compiled: March 2026 | Africa AI Atlas — 7Square Inc.**

## Scoring Guide

- **AI Use Cases**: Estimated count of documented, active AI use cases (Low: 1-5, Medium: 6-15, High: 16-30, Very High: 30+)
- **Public Sector AI**: None / Nascent / Developing / Established / Advanced
- **Financial AI**: None / Nascent / Developing / Established / Advanced
- **Agriculture AI**: None / Nascent / Developing / Established / Advanced
- **Health AI**: None / Nascent / Developing / Established / Advanced
- **Sector Diversity**: Number of sectors with active AI deployments (out of ~8 key sectors: finance, health, agriculture, education, energy, transport, government, security)
